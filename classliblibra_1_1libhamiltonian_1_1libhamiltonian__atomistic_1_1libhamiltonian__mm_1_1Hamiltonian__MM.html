Create many-body interaction. One many-body interaction can include the interaction of N particles at once. It can be factorized down to 2, 3, or 4-body interactions - e.g. for acceleration, or it may not be easily factorizable (e.g. lattice sums) This functions sets the internal pointers to point to the external coordinates (also parameters, forces, etc). And the internal parameters will be set up for this particular interaction.

Parameters
[in]tType of the MM functional to set for this interaction.See void Hamiltonian_MM::set_interaction_type_and_functional() for details
[in]pFunctional of given type for this interaction. See void Hamiltonian_MM::set_interaction_type_and_functional() for details
[in]szThe number of atoms involved in the many-body interactions
[in]idThe pointer to the array of IDs of the atoms invloved in the interaction
[in]rThe pointer to the array of addresses of external atomic coordinates. Change of those coordinates will affect the computations with the same interaction object.
[in]gSame as r, but for fragmental coordinates. Change of these will affect only coarse-grained potentials
[in]mSame as r, but for fragmental coordinates. Change of these will affect only coarse-grained (molecular-level) potentials
[in]qThe pointer to the array of addresses of external atomic charge. Change of those charges will affect the computations with the same interaction object.
[in]epsilonThe pointer to the array of addresses of atomic vdw interaction strengths
[in]sigmaThe pointer to the array of addresses of atomic vdw radii
[in]nexclThe number of exclusions in the system
[in]excl1The pointer to the array of atomic indices for the first atom in the exclusion pair
[in]excl2The pointer to the array of atomic indices for the second atom in the exclusion pair
[in]scaleThe pointer to the array of scaling constants for each exclusion pair
[in]displr2The pointer to the array of addresses each pointing to the variable that holds the square of the atomic displacement for each atom
[in]displT_2The address of the variable that holds the square of the unit cell translation vector fluctuation
[in]paramsThe dictionary of parameters to set to this ineraction. Possible options depend on the type of many-body interaction

Functional type Possible params keys Meaning

mb R_on Distance when the switching function starts (and is = 1) R_off Distance when the switching function stops (and is = 0) R_on2 Square of R_on R_off2 Square of R_off is_cutoff The flag wheter the cutoff is used (if not - the full range is applied)

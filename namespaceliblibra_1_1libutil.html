int liblibra::libutil::delta ( vector< int > &  A,

int &  a,
int &  b

This function searches for pair of indices in A and B which are different, they are then assigned to a and b correspondingly All other indexes in A and B should be equivalent (not necessarily in the same order)

The function returns 1 if only 1 pair of indices is different between A and B, return 0 otherwise

That is res = 1 means A and B are coupled, while res = 0 - are not

Algorithm: first construct the list of overlapping orbitals second - go through this list and determine the populations of each orbital in this list in each of the configurations

THIS VERSION REGARDS SIGN OF THE ORBITAL INDEX

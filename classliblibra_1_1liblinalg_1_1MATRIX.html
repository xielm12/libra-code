void liblibra::liblinalg::MATRIX::Rotation ( const VECTOR &  u)

This function initializes the matrix to a rotation matrix for rotation around the axis given by direction of the vector: (u/|u|) on amount given by norm of vector u: |u|. If the vector has a zero length - this will be an identity matrix

Here is efficient implementation of Rodrigues formula: M = I + W * sin_psi + W*W*(1.0 - cos_psi) where I - identity matrix | 0 -n.z n.y | W = skew(n) = | n.z 0 -n.x | |-n.y n.x 0 |

Create a new AO (STO-3G) object and add it to existing (including empty) list of orbitals (basis)

Parameters
[in]Atom_nameThe name of the atom (element) for which we want to create AO
[in]RThe coordinate at which to create the atomic orbital.
[in]Atom_shellThe name of the AO shell e.g. 1s, 2p, 3p, 3d, etc.
[in]NzetaThe number of different Slater-type zetas - in most cases 1, for d-orbitals can be 2
[in]NquantThe principal quantum number of the orbital
[in]IPThe state-specific ionization potential (not actually used in this function!)
[in]exp1The first Slater-type exponent
[in]exp2The second Slater-type exponent - is not used when Nzeta = 1
[in]coeff1The contraction coefficient of the first Slater zeta orbital
[in]coeff2The contraction coefficient of the second Slater zeta orbital (is effectively = 0 if Nzeta = 1)
[in,out]basis_aoThe list of AO objects - the basis we are creating

Sequential application of this function with proper parameters will generate the atomic basis - the list of AO objects

For conversion of the spherical hamonics to Cartesian coordinates see e.g.: http://csi.chemie.tu-darmstadt.de/ak/immel/script/redirect.cgi?filename=http://csi.chemie.tu-darmstadt.de/ak/immel/tutorials/orbitals/hydrogenic.html So the z2 orbital is actually: (1/(?)) *[ 2*z2 - (x2+x2) ] and the x2-y2 orbital is (1/sqrt(2)) * [x2 - y2]

An auxiliary function that returns parameters of STO-3G d-type orbitals

Parameters
[in]NquantThe principal quantum number of the atom for which we construct the AO
[out]alpThe exponents of the primitive Gaussians in the STO-3G expansion
[out]coeffThe contraction coefficients the primitive Gaussians enter with in the STO-3G expansion

The function takes expansion coefficients to be equal to the expansion coefficients fron the STO-3G basis in the valence shell. The alphas are obtained from a scaling law. Data are taken from EMSL website (mostly) or extrapolated

Nquant = 1 - H exp is taken directly from Pople: Hehre, Stewart, Pople JCP 1969, 51, 2657 Nquant = 2 - C exp is taken directly from Pople: Hehre, Stewart, Pople JCP 1969, 51, 2657 Nquant = 3 - taken to be the same as for Nquant = 4 Nquant = 4 - Sc exp is calibrated to zeta = 1.7 (exponent with the largest weight) Nquant = 5 - Nb exp is calibrated to zeta = 1.64 There is no STO-3G bases for 6-row elements, so will need to extrapolate so far - just keep it the same as for 5-row

This function computes the overlap of two Slater determinants sd_i and sd_j

Parameters
[in]sd_i,sd_j: Slater Determinants at possibly different time steps (or at the same one) The computed overlap value will be returned

<SD(t)|SD(t')>. = det | <psi_i(t) | psi_j(t') > |

This is based on the Lowdin formula. See, for instance: (1) Ryabinkin, I. G.; Nagesh, J.; Izmaylov, A. F. Fast Numerical Evaluation of Time-Derivative Non-Adiabatic Couplings for Mixed Quantum-Classical Methods. J. Phys. Chem. Lett. 2015, 6, 4200–4203.

Update the derivative coupling matrix (in AO basis): <AO(i)|d/dR_c|AO(j)>

Parameters
[in]x_periodThen number of periodic shells in X direction: 0 - only the central shell, 1 - [-1,0,1], etc.
[in]y_periodThen number of periodic shells in Y direction: 0 - only the central shell, 1 - [-1,0,1], etc.
[in]z_periodThen number of periodic shells in Z direction: 0 - only the central shell, 1 - [-1,0,1], etc.
[in]t1The periodicity vector along a crystal direction ("X")
[in]t2The periodicity vector along b crystal direction ("Y")
[in]t3The periodicity vector along c crystal direction ("Z")
[in]atom_to_ao_mapThe mapping from atomic indices to AO indices: atom_to_ao_map[n][i] - is the global index of i-th AO on the n-th atom
[in]ao_to_atom_mapThe mapping from the global AO index to the atomic index: ao_to_atom_map[I] - is the index of the atom on which AO with the global index I is located
[in]basis_aoThe list of all AOs (basis)
[in]cThe index of the nuclear DOF (atom index) to which the electronic states are coupled - only this derivative matrix is considered here
[out]Dao_xThe derivative coupling matrix in AO basis, coupled to x coordinate of the atom with index c
[out]Dao_yThe derivative coupling matrix in AO basis, coupled to y coordinate of the atom with index c

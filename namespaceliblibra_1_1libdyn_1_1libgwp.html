This function computes a derivative coupling between two moving Gaussians <G_1|d/dr|G_2>, where:

G_a(r; R_a, P_a, alp_a, gamma_a) = (2*alp_a/pi)^(Ndof/4) * exp(-alp_a*(r-R_a)^2 + i*(P_a/hbar)*(r-R_a) + i*gamma_a/hbar)

Look derivations at: https://github.com/alexvakimov/Derivatory/blob/master/Gaussian_wavepackets.pdf

Parameters
[in]R1,R2Multidimensional vectors containing the components of position of the multidimensional Gaussians. Assumed to be Ndof x 1 vectors
[in]P1,P2Multidimensional vectors containing the components of momenta of the multidimensional Gaussians. Assumed to be Ndof x 1 vectors
[in]gamma1,gamma2The phase factors of the overall Gaussians
[in]alpThe Gaussian width factor. Assumed to be the same for all components of both multidimensional Gaussians
[in]hbarThe Planck constant in selected units

The function returns the derivative coupling vector - a complex vector

The metastable potential - e.g. see Donoso, A.; Martens, C. Quantum Tunneling Using Entangled Classical Trajectories. Phys. Rev. Lett. 2001, 87, 223202. The default parameters are chosen to make 2 bound state, so the system is highly quantum-mechanical

[out]d2HThe pointer to the matrix in which the 2-nd order derivatives of the Hamiltonian will be written
[in]paramsThe model parameters: can be up to 3 parameters (see the chart below). Otherwise, the default values will be used:

Internal parameter Input Default value m param[0] 2000.0 w param[1] 0.010 b param[2] 0.2981

Cubic hamiltonian and its derivatives in diabatic representation:

H_00 = 0.5*m*w^2 * x^2 - (b/3) * x^3

The symmetric double well potential. Two local minima are located at x = -1.0 and 1.0. A local maximum is located at x = 0.

Double well potential Hamiltonian and its derivatives in diabatic representation:

H_00 = A * ( 0.25*x^4 - 0.5*x^2 )

explicit functions

f(x) = A * [ (1/4)*x^4 - (1/2)*x^2 ] f'(x) = A * [ x^3 - x ] f''(x) = A * [ 3x^2 - 1 ]

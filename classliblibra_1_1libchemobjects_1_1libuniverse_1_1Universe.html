Adding (chemical) Element object to the Universe object.

Parameters
[in]recThe input Element object to be added as a record in the Universe object

This is user-interface function. It checks for existance of the Element rec in array PeriodicTable. If this element exists

  • do nothing; if it does not - add it to array Returns 1 if this is a new record (it just has been added) and 0 if record existed in array (we did not add it this time). Returns -1 if there is an error.

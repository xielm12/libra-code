std::string  path,
int &  status 
)

Load the state of the Element object from a property tree

Each data member found in the property tree is extracted as the member of the Barostat object. The status of each found data member is set to 1.

Parameters
[in]ptThe property tree from which the properties of the Element will be extracted
[in]pathThe parameter controlling from which level of the tree we try to extract the Element object
[out]statusIs the global status of the success of the operation. It is 1 is at least one Element member is found at given level of the property tree.

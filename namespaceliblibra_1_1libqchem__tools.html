To compute the charge densities for the combinations of fragment MOs

Parameters
[in]CThe matrix containing the transformation coefficients = in which combintation to take the MOs of the sub-systems
[in]hamIs the list of Hamiltonian objects (of type listHamiltonian), each containing both electronic structure (MO) information and the basis (AO) information about each sub-system
[in]systThe nuclear structure of the system (combined)
[in]active_orbHolds indices of the fragment MOs each fragment contributes to the total pool of the fragment MOs
[in]prmsParameters controlling how to execute the calculations

This function compute wfc of a given state (orbital) and prints it in CUBE format for visualization See about Gaussian CUBE format at: http://paulbourke.net/dataformats/cube/

prms.charge_density_prefix - specifies the directory to which the files will be written prms.orbs - the indices of the MOs to print

double liblibra::libqobjects::PrimitiveG::normalization_factor ( )

Compute normalization factor for given primitive Gaussian

double nom = pow((8.0*G_alpha),(x_exp+y_exp+z_exp))*FACTORIAL(x_exp)*FACTORIAL(y_exp)*FACTORIAL(z_exp); double denom = FACTORIAL(2*x_exp)*FACTORIAL(2*y_exp)*FACTORIAL(2*z_exp); res = pow((2.0*G_alpha/M_PI),0.75)*sqrt(nom/denom);

Meaning: if N - is a result of this function and G(l,m,n,alp) = x^l * y^m * z^n * exp(-alp*r^2) a primitive Gaussian then g = N * G(l,m,n,alp) - is normaized: integral(g,g) = 1.0

= 1/sqrt(<G(A)|G(A)>)

int liblibra::libqobjects::PrimitiveG::x_exp

The class that represents Gaussian primitive functions of arbitrary angular momentum and exponent.

f = (x-X)^{x_exp} * (y-Y)^{y_exp} * (z-Z)^(z_exp) * exp(-alpha * (r - R)^2) R = (X, Y, Z)^T

Compute the matrix element of the derivative coupling <G(A)|d/dR_B|G(B)> note that the d/dR_B uses the derivative w.r.t. the coordinates of the primitive Gaussian B - otherwise it would be zero

Parameters
[in]GAOne primitive Gaussian
[in]GBSecond primitive Gaussian
[in]is_normalizeThe flag telling whether we need to normalize the result: is_normalize = 1 - the overlap will be normalized (Gaussians are assumed to be unnormalized), is_normalize = 0 - no need for normalization (Gaussians are assumed to be normalized)
[in]is_derivsif = 1 - also compute the derivatives of the overlap w.r.t. coordinates of each primitive center and w.r.t. the pseudopotential center
[out]dMdAThe derivative of the derivative coupling vector w.r.t. the coordinates of the primitive Gaussians A center (if is_derivs = 1)
[out]dMdBThe derivative of the derivative coupling vector w.r.t. the coordinates of the primitive Gaussians B center (if is_derivs = 1)
[in,out]auxdThe list of the pointers to pre-allocated pieces of memory (for variables of the double type)
[in]n_auxThe length of the array to which each of the auxd[i] pointers points.

Compute the matrix element of the electron repulsion integral for arbitrary primitive Gaussians: <G(A)G(B)|V|G(C)G(D)>, where V(r) = 1/|r1-r2|

Parameters
[in]GAPrimitive Gaussian, #1
[in]GBPrimitive Gaussian, #2
[in]GCPrimitive Gaussian, #3
[in]GDPrimitive Gaussian, #4
[in]is_normalizeThe flag telling whether we need to normalize the result: is_normalize = 1 - the overlap will be normalized (Gaussians are assumed to be unnormalized), is_normalize = 0 - no need for normalization (Gaussians are assumed to be normalized)
[in]is_derivsif = 1 - also compute the derivatives of the overlap w.r.t. coordinates of each primitive center
[out]DAThe derivative of the integral w.r.t. the coordinates of the first (A) primitive Gaussian (if is_derivs = 1)
[out]DBThe derivative of the integral w.r.t. the coordinates of the second (B) primitive Gaussian (if is_derivs = 1)
[out]DCThe derivative of the integral w.r.t. the coordinates of the third (C) primitive Gaussian (if is_derivs = 1)
[out]DDThe derivative of the integral w.r.t. the coordinates of the fourth (D) primitive Gaussian (if is_derivs = 1)
[in,out]auxdThe list of the pointers to pre-allocated pieces of memory (for variables of the double type)
[in]n_auxThe length of the array to which each of the auxd[i] pointers points.
[in,out]auxvThe list of the pointers to pre-allocated pieces of memory (for variables of the VECTOR type)

Compute the overlap of 3 arbitrary primitive Gaussians: <G(A)|G|G(B)> Note that specific cases of such integral include dipole, multipole moments, and components of pseudopotential: <G(A)|R_c|G(B)>, <G(A)|R_c * R'_c|G(B)>, <G(A)|exp(-alp*(r-R_c))|G(B)>, etc.

Parameters
[in]GAOne primitive Gaussian
[in]GThe function that defines the momoent - this function is assumed in the form of generalized Gaussian
[in]GBSecond primitive Gaussian
[in]is_normalizeThe flag telling whether we need to normalize the result: is_normalize = 1 - the overlap will be normalized (Gaussians are assumed to be unnormalized), is_normalize = 0 - no need for normalization (Gaussians are assumed to be normalized). The normalization only uses the factors for GA and GB!
[in]is_derivsif = 1 - also compute the derivatives of the overlap w.r.t. coordinates of each primitive center
[out]dIdAThe derivative of the integral w.r.t. the coordinates of the first (A) primitive Gaussian (if is_derivs = 1)
[out]dIdRThe derivative of the integral w.r.t. the coordinates of the moment-defining primitive Gaussian (if is_derivs = 1)
[out]dIdBThe derivative of the integral w.r.t. the coordinates of the second (B) primitive Gaussian (if is_derivs = 1)
[in,out]auxdThe list of the pointers to pre-allocated pieces of memory (for variables of the double type)
[in]n_auxThe length of the array to which each of the auxd[i] pointers points.

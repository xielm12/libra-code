=========== Look in: mEigen_eigensolve.cpp ================== Solving the generalized eigenvalue problem: H * C = S * C * E

Solve the generalized eigenvalue problem: H * C = S * C * E i-th column of C contains i-th MO (coefficients of expansion in terms of AOs) C[i][j] - is the weight of j-th AO in i-th MO More, generally C.col(i) is the eigenvector corresponding the eigenstate E_i

int symm - is the flag for H and S matrices symmetrization: symm = 0 - don't symmetrize (use the input matrices as they are) symm = 1 - do symmetrize them

H * C = (N_bas x N_bas) x ( N_bas x N_mo ) = N_mo x N_mo

S * C * E = (N_bas x N_bas) x (N_bas x N_mo ) x (N_mo x N_mo) = N_bas x N_mo

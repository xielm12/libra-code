This operator performs in-plane rotation - changes 2 components of an arbitrary vector

Parameters
[in,out]xThe x component of 2D vector
[in,out]yThe y component of 2D vector
[in]phiThe angle of rotation (radians)

Application of the operator exp(iL * phi) = exp (phi * (x * d/dy - y * d/dx) ) to vector (x,y)^T: 

          | x |     |  cos(phi)   -sin(phi)  |    | x |

exp(iL * phi) | | = | | * | | | y | | sin(phi) cos(phi) | | y |

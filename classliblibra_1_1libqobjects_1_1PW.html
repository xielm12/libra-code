void liblibra::libqobjects::PW::complete ( )

Complete the PW object by adding the complex conjugate part This is the storage saving trick, due to c(-G) = conj(G), which is usually applied only to gamma-point (k_point==0). We also assume that the very first coefficient coeff[0] corresponds to G-point = 0,0,0 This is true for QE wavefunctions

The function changes the original object

double liblibra::libqobjects::PW::energy

energy of this orbital (eigenvalue)

This class describes the data type for storing general MO in the plane wave basis

psi_{i,k}(r) = 1/sqrt(V) sum c_i(G) * exp(-i(G+k)*r), where we run over all G points in the expansion G

this wavefunction accounts both the band index (i) and the k-point index (k)

read_basisset_file: reads a basis set from fileName. The basis set exponents and contraction coefficients are placed in result. The reading procedure is bit convoluted because the basis set file follows the Fortran syntax, with wrapping and skipping empty elements. We basically need to emulate fortran `format' statement. There is one "but" though: AhlrichsDenFit does not follow the format syntax so it will/may be misread by eg. dalton. What a mess...

The parser is implemented as a state machine. It still cannot read ANO-type basis sets...
